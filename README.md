"# Meus Gastos
		Um aplicativo de organização financeira desenvolvido com **React Native** usando **Expo** e serviços do **Firebase**.
# Tecnologias
	- React Native
	- Expo
	- Firebase (Auth, Firestore, etc.)
- ## Funcionalidades
  - Organização e controle de gastos pessoais
  - Integração com Firebase para armazenamento e autenticação
- ## Instalação
	- 1. Clone o repositório:
	-  ```bash git clone [URL_DO_REPOSITORIO] ```
	-    2. Copie o arquivo de exemplo `.envexemple` e renomeie para `.env`:
	-       ```bash cp .envexemple .env ```
	-  3. Preencha as informações do Firebase no arquivo `.env`.
	-  4. Instale as dependências e rode o projeto:
	-      ```bash npx expo start ```
- ## Dependências
	- Firebase (somente serviços utilizados no app)##
