"# Meus Gastos\n\nUm aplicativo de organização financeira desenvolvido com **React Native** usando **Expo** e serviços do **Firebase**.\n\n
## Tecnologias\n
- React Native\n
- Expo\n
- Firebase (Auth, Firestore, etc.)\n\n
- ## Funcionalidades\n
  - Organização e controle de gastos pessoais\n-
  - Integração com Firebase para armazenamento e autenticação\n\n
- ## Instalação\n
- 1. Clone o repositório:\n
   ```bash\n
      git clone [URL_DO_REPOSITORIO]\n
      ```\n
-    2. Copie o arquivo de exemplo `.envexemple` e renomeie para `.env`:\n
-       ```bash\n
-       cp .envexemple .env\n
-      ```\n
-  3. Preencha as informações do Firebase no arquivo `.env`.\n
-  4. Instale as dependências e rode o projeto:\n
-      ```bash\n
-     npx expo start\n
-    ```\n\n
-    ## Dependências\n
-    - Firebase (somente serviços utilizados no app)\n\n##
